# Open Market Architecture Study


## 브랜치 전략
모놀리식 스타일은 master로 둔다.
MSA 스타일은 MSA 브랜치로, DDD 스타일은 DDD 브랜치로 둔다

## 구성 요소
- 유저 ( 구매자 / 판매자 )
- 주문
- 배송
- 상품
- 결제

## 기본 시나리오
- Only REST API 로 통신

## End Point

### 유저
- 유저 가입 
- 유저 탈퇴
- 유저 정보 수정
- 유저 정보 보기

### 상품
- 상품 등록
- 상품 등록 해제
- 상품 정보 수정
- 상품 정보 보기

### 배송
- 배송 상태 조회 
- 배송 상태 업데이트

### 주문
- 주문 하기
- 주문 취소
- 주문 조회

### 결제
- 결제 하기
- 결제 취소
- 결제 내역 보기
